Exercise 1 – Employee Management System Setup

What is Spring Boot and Spring Data JPA
Spring Boot is an opinionated framework built on top of the Spring ecosystem that simplifies the configuration and bootstrap of enterprise applications. Spring Data JPA is part of the Spring Data project that makes it easy to implement JPA-based repositories, reducing the boilerplate code required to interact with database schemas.

Dependencies Used
- Spring Data JPA: Core ORM mapping library.
- Spring Web: Provides REST web services support.
- H2 Database: In-memory relational database.
- Lombok: Auto-generates standard getters, setters, constructors, and builders at compilation time.
